# SportPlay

SportPlay is a Django-based web application where users can browse sports games and register for the games they want to participate in.

## Features
- User registration and login
- Browse available games
- Game details with images
- Register for games
- Cancel registration
- View personal registered games
- Search games
- Admin panel to manage games

## Technologies Used
- Python
- Django
- HTML / Django Templates
- SQLite
- Git & GitHub
